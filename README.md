# Portfolio

## Overview

Welcome to my portfolio project! This project acts as a dynamic showcase,
skillfully presenting personal information for an effective job search. It
operates as an interactive resume, enabling me to show my experience and recent
projects with an engaging user experience.

## Features

- Personal information display
- Project showcase
- Skills and expertise presentation
- Form for contacting by email
- Responsive design for optimal viewing on various devices
- Animation and transitions
- Dynamic header and light/dark theme
